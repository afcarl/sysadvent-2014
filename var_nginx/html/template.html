Templating Example

This page is rendered using lua-resty-template. This is not included with openresty but is installed using the lua package manager luarocks

In this file: 

/var/nginx/html/template.html
you can see the familiar handlebars style markup

This is much easier than complicated chunks of lua heredoc and ngx.say() chunks

Some nginx data

  • Current time (ngx.var.utctime): 
    {[ngx.utctime()]}
  • Variable passed in from lua: 
    {{message_string}}


  local template = require 'resty.template'
  template.caching(false)
  template.render("template.html", { ngx = ngx, message_string = "I passed this in from resty_template.lua" })
